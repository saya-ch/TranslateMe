# 安全边界 · 帮我说出口

本文档固化产品的安全边界，所有生成逻辑、界面文案、测试用例都必须符合本文档。

## 一、产品边界（绝对不做）

1. 不做心理诊断。
2. 不输出抑郁症、焦虑症等病名作为判断或标签。
3. 不做心理测评或分数评估。
4. 不判断严重程度。
5. 不承诺治疗效果。
6. 不替代心理老师、心理咨询师、医生或学校已有流程。
7. 不做长期 AI 陪聊。
8. 不收集真实姓名、学校、班级、手机号。
9. 不保存敏感输入。
10. 高风险表达必须直接切换到真人帮助和安全支持页。

## 二、术语边界说明

"抑郁""焦虑"等词的使用边界：

- **可以出现**：在本安全边界文档、项目背景文档、界面边界声明中，用来解释"本工具不做诊断"。
- **不可以出现**：在生成给用户的行动卡中，作为判断或标签出现。例如不得输出"你这是抑郁""你可能患有焦虑症"等。

## 三、高风险表达检测

### 3.1 触发词列表

输入包含以下表达或近似表达即触发：

想死、不想活、自杀、自残、伤害自己、伤害别人、结束一切、永远消失、活着没意思、撑不下去、想跳楼、想割腕

以及常见变体：不想活了、寻死、轻生、了结自己、活不下去、撑不住、不想存在、消失算了等。

### 3.2 检测策略

- 输入预处理：去空格、转小写、繁简统一。
- 关键词子串匹配 + 少量正则覆盖变体（如"想*死""不*活""想*跳楼"）。
- 命中即返回高风险标记，不返回具体命中的词给前端展示，避免反向提示。

### 3.3 触发后行为

- 立即停止普通卡片生成流程，不调用 generator。
- 显示安全支持页。
- 不询问危险方式、地点、计划、时间等细节。
- 不输出任何危险行为方法。
- 提供"返回首页"按钮，返回后清空输入，不留存。

## 四、安全支持页文案规范

安全页严格区分两类资源，不混称"紧急电话"：

### 心理支持与求助咨询
- 12355 青少年服务热线
- 12356 心理援助热线
- 联系身边可信成年人、紧急联系人、学校心理老师

### 如存在立即人身危险
- 请拨打 110（报警）/ 120（急救）

文案示例：

> 这可能需要立即的真人帮助。
> 如果你或身边的人正处于立即危险中，请立刻联系身边可信成年人、紧急联系人或学校心理老师。
> 可以拨打 12355 青少年服务热线或 12356 心理援助热线，用于心理支持与求助咨询。
> 如存在立即人身危险，请拨打 110 / 120。
> 这个工具不能替代专业人员，也不会继续分析危险细节。

## 五、生成层安全

- 卡片文案来自预审模板，初赛阶段不调用 LLM，用本地规则和模板模拟受控 AI 转译。
- 模板中禁止出现：作为判断或标签的诊断词（抑郁/焦虑/双相/ADHD 等）、严重度词（轻度/中度/重度/危机）、承诺治愈词（一定会好/我能治好）。
- 模板中禁止出现：危险行为方法、对家长的指责性表述。
- 用户输入仅用于"复述"环节，复述前做二次安全词过滤，防止用户输入被原样回显成危险内容。

## 六、隐私边界

- 无登录、无后端、无数据库。
- 应用内不主动收集姓名、学校、班级、手机号等身份信息。
- 不上传、不保存用户输入；输入仅存于组件 state，离开页面或刷新即销毁。
- 复制功能走浏览器原生 Clipboard API，不经过任何服务端。
- 部署说明：若部署到 GitHub Pages / Vercel / Netlify，托管平台可能产生基础访问日志，这部分由平台方管理，不在应用控制范围内。

## 七、界面常驻边界声明

每个页面底部固定一行：本工具不做心理诊断、不替代专业人员；需要心理支持可拨打 12355 / 12356；如存在立即人身危险请拨打 110 / 120。

## 八、后续 LLM 接入约束

初赛 Demo 用本地规则和模板模拟受控 AI 转译。后续产品方向可接入受安全规则约束的 LLM，但必须满足：

1. 先做高风险拦截（本文件第三节），命中即跳安全页，不进入 LLM 生成。
2. LLM 输出必须经过非诊断审查，不得出现病名标签、严重度词、治愈承诺。
3. 禁止长期陪聊和多轮对话，单次生成即结束。
4. 不得输出危险行为方法。
5. 不得收集或上传用户输入到外部服务。
